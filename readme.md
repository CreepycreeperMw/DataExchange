# MB-DEX
Minecraft Bedrock DEX _(short for Data Exchange)_ is both an API and Protocol
to efficiently **communicate data** between addons using Script API.

The API is built to be
- ⚡ **blazingly fast**
- ⚙️ **efficient**
- 🧠 **straight forward**
- 🔁 **backwards compatible**
